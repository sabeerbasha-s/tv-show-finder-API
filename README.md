# tv-show-finder-API

TV Show Finder is a web application built using HTML, CSS, and JavaScript. It allows users to search for TV shows and view details such as show name, language, rating, release date, and poster using the TVMaze API.

## Features

- Search TV Shows
- Show Name
- Language
- Rating
- Release Date
- Show Poster
- Uses TVMaze API

## Technologies Used

- HTML
- CSS
- JavaScript
- TVMaze API
